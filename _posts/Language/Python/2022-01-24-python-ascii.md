🔔 문자를 아스키코드로 변환하고 아스키코드를 문자로 변환하는 과정을 알아보겠습니다.

## 1️⃣ 문자 -> 아스키코드 변환
ord()
```python
print(ord("A"))

>>> 65
```
<br>

## 2️⃣ 아스키코드 -> 문자 변환
chr()
```python
print(chr(65))

>>> A
```

<br>
