## 문제

```html
0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.
```


## 입력  
```html
첫째 줄에 정수 N(0 ≤ N ≤ 12)이 주어진다.
```

## 출력 
```html 
첫째 줄에 N!을 출력한다.
```


## 예제 입력
```python
10
```

## 예제 출력
```python
3628800
```

<br>

## 해결

반복문 대신 재귀함수로 팩토리얼을 구현할 수 있습니다.  

```python
def factorial(n):
    result = 1
    if n > 0:
        result = n * factorial(n-1)
    return result


n = int(input())

print(factorial(n))
```
